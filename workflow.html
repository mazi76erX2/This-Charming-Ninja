My Workflow for Django

In command prompt

  1. cd to the folder you want to work in.

  2. pip install virtualenv
        virtualenv mysiteENV

  3. 'mysite\scripts\activate.dat'

  4. 'pip install django'

then 

'pip install psycopg2'

or whatever database package you’re using.

  1. ‘python -m django –version’ to make sure django is installed

  2. ‘django-admin startproject ‘ then ‘cd ‘

  3. python manage.py runserver 8080 (just incase 8000 is being used). This instruction is to check that the setup is good.

  4. ‘python manage.py createsuperuser’ to create admin

  5. ‘python manage.py startapp ‘ to make app inside main app

  6. make a python file /urls.py. Make edits to urls in each and and edits in views.

    ‘url(r’^/’, include(‘.urls’)),’ in /urls.py

  7. If you’re using sqlite3 leave the settings.py file the way it is. If you’re using postgres (for example) add:

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql_psycopg2',
        'NAME': 'myproject',
        'USER': 'myprojectuser',
        'PASSWORD': 'password',
        'HOST': 'localhost',
        'PORT': '',
    }
}

  1. make models for using ‘https://docs.djangoproject.com/en/1.11/ref/models/instances/#django.db.models.Model‘ in /models.py
for example:

  1. Activate models by adding ‘.apps.Config’ into INSTALLED_APPS in the settings file.

  2. ‘python manage.py migrate’ to setup up database

  3. ‘python manage.py makemigrations ‘ Tells django to adds the changes to the models to the database.

  4. ‘python manage.py sqlmigrate 0001’ Shows you all the changes that will be made to the database.

NOTE: Whenever you make changes to the model you must makemigrations then migrate (Because you are changing how the database saves the data).

  1. ‘python manage.py migrate’ Again to actually make the changes.

  2. Make the app modifiable in the admin by adding

Eventually change to:

‘urlpatterns = [ url(r’^$’, _list, name=’list’), url(r’^create/$’, _create), url(r’^(?P\d+)/$’, _detail, name=’detail’), url(r’^(?P\d+)/edit/$’, _update, name=’update’), url(r’^(?P\d+)/delete/$’, _delete), #url(r’^s/$’, “.views.“), ]’
